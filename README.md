# Supply Chain Performance & Delivery Analytics Dashboard

## 📌 Project Overview
This project focuses on analyzing sales performance and delivery efficiency using an Excel-based dashboard. The goal was to identify trends, monitor key metrics, and highlight areas affecting delivery performance.

## 🛠️ Tools Used
- Excel
- Power Query
- Pivot Tables
- Charts & Dashboard Design

## 📊 Key Features
- Cleaned raw data by removing unnecessary columns and fixing formats
- Created KPIs such as Total Orders, Total Revenue, and Late Deliveries
- Built interactive Pivot Tables and charts for analysis
- Added slicers for dynamic filtering and better user interaction
